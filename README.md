# I&I(Inventory and Invoice manager)
Fullstack business product inventory manager and invoice generator.

# Description
A full stack application with a RESTful express API backend and a React, built from scrath, front-end. 
Complete with user authentication + persistent login via access/refresh JWT tokens.

# Technologies Used
- React
- Node.js
- Javascript
- CSS
- REST API
- Express
- JWT
- MongoDB

# Features
 - Persistent login, user authentication via long persisting refresh token cookies and quick expiration memory only access tokens.
 - Auto inventory adjustment on invoice creation.
 - Auto adjusting input based on inventory limit and set prices.
 - Invoice generator into fixed A4 printable pages.

# Future features
- Mobile responsiveness.
- Night mode.
- Send invoice via email on request.
- Autofill state/city/zip based on address.
